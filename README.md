# Habit Tracker 🚀

A modern, Duolingo-inspired Habit Tracking application built with Kotlin and Jetpack Compose (View-based with Material 3). This app helps you build better routines with a clean, engaging interface and rewarding milestones.

## ✨ Features

- **Dynamic Dashboard**: View your daily progress at a glance with a large circular indicator and motivational quotes that change every launch.
- **Habit Categories**: Organize your life with categories like Health, Work, Personal, Finance, and more.
- **Milestones**: Unlock achievements like "Centurion" (100 completions) or "Yearly Streak" (365 days) as you progress.
- **Duolingo Theme**: A vibrant, modern UI using Duolingo-inspired green and white aesthetics.
- **Local Persistence**: Powered by Room Database for fast, offline-first data management.
- **Secure Sessions**: Simple login and session management to keep your habits personal.
- **Performance Insights**: Track your overall completion rates and best streaks.

## 🛠 Tech Stack

- **Language**: Kotlin
- **Database**: Room
- **Architecture**: MVVM (Model-View-ViewModel)
- **UI Components**: Material 3, ViewBinding, Navigation Component
- **Processing**: KSP (Kotlin Symbol Processing)
- **Concurrency**: Coroutines & Flow

1. Clone the repository:
   ```bash

   ```
2. Open the project in **Android Studio Ladybug** or newer.
3. Build and run on an emulator or physical device (API 29+).
